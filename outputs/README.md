图1：关键词聚类共现网络图（知识基础图）
一、本关键词聚类图谱依托2010-2024年WOS收录的宽禁带GaN器件领域文献构建，通过关键词聚类划分知识群组，厘清该领域的知识基础构成、细分研究分支与学科底层理论来源，从关键词关联与聚类分布，拆解支撑GaN器件研究的经典理论、核心技术方向与学科依托体系。
二、主要发现：图谱模块化Q=0.2728、轮廓系数S=0.6728＞0.5，聚类结果可信，领域划分为：#0电流崩塌、#1高效率、#2逻辑门、#3高电子迁移率晶体管四大核心知识集群：#3高电子迁移率晶体管（HEMT）是领域最核心知识根基，AlGaN/GaN异质结、二维电子气等关键词构成全领域底层理论；#0电流崩塌是器件可靠性方向基础，陷阱、漏电、击穿电压为该分支核心研究基础；#1高效率聚焦功率器件能效优化，宽禁带半导体、功率变换器是产业化落地的知识依托；#2逻辑门面向射频数字器件，二极管、欧姆接触构成微电子应用知识体系；聚类间大量交叉连线证明四大分支共用GaN材料基础理论，学科交叉特征显著。
三、图谱局限:第一，CiteSpace自动分词无法合并同义词，如“HEMT”与“高电子迁移率晶体管”、缩写与全称被拆分为独立节点，造成同类知识碎片化；第二，筛选阈值限定后，冷门细分方向关键词被剔除，小众前沿分支的知识基础无法体现；第三，仅依托英文WOS数据库，中文文献关键词未纳入，国内本土特色研究方向的知识根基缺失；第四，关键词仅反映文献表层主题，无法区分理论基础与应用技术的层级差异。


图2：关键词时间线演化图（趋势演化图）
一、本图回答的研究问题本时间线图谱以2010-2024年时间轴为载体，追踪各聚类关键词从萌芽、兴起至成熟的时序变化，梳理GaN器件领域研究热点更迭规律、分阶段演化脉络，回答作业命题“研究趋势如何演化”，从突现关键词的时间分布，划分领域发展阶段，预判未来前沿发展方向。<br/> <br/>
二、主要发现领域演化分为三个阶段：①2010-2015萌芽奠基期：#0电流崩塌、GaN材料、陷阱缺陷等关键词集中爆发，领域聚焦器件可靠性基础机理，是全领域理论奠基阶段；②2015-2020高速发展期：#3高电子迁移率、#1高效率关键词快速兴起，研究从基础缺陷转向器件性能优化，功率变换器、高频应用成为热点；③2020-2024产业化深化期：#2逻辑门、热阻、射频放大器成为前沿，研究从单器件机理拓展到集成电路与热管理应用。#0电流崩塌全周期贯穿始终，是领域永恒研究痛点；#4热阻为后期新兴聚类，代表散热方向是当下前沿增长点；不同聚类的连线随时间逐步密集，体现各研究分支不断融合。
三、图谱局限:第一，数据库文献收录存在1-2年滞后，2023-2024年最新前沿文献收录不全，前沿趋势预判存在偏差；第二，时间切片设置固定，无法精准捕捉短周期突发热点，部分持续周期不足2年的新兴关键词无法形成有效时序节点；第三，算法依赖发文频次判定热点，发文少但突破性强的前瞻技术关键词无法被标记，低估小众前沿发展潜力；第四，未区分发文量与成果质量，部分量产普及型关键词仅因发文多被判定为热点。


图3：作者合作网络图（领域推动主体图）
一、本作者合作网络图基于2010-2024年WOS作者合著数据构建，量化解析领域科研主体分布、团队协作结构与国内外研究力量布局，明确哪些学者与科研团队主导GaN器件领域研发，厘清国内外协作差异，精准回应作业命题“谁在推动这个领域发展”，从作者合作视角锚定领域核心推动力量。

三、图谱局限:第一，制图筛选阈值k=5，仅保留发文≥5篇作者，剔除发文量少但成果突破性强的新锐青年学者；第二，数据源仅限WOS英文文献，CNKI中文文献未收录，大量国内本土高产中文成果无法入图；第三，仅依靠合著频次表征合作强度，无法区分短期论文合作与长期项目合作，不能体现合作成果学术质量；第四，外文译名不统一、同名不同人无法被算法识别，存在作者节点错误合并问题。
